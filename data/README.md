# Data directory

Place raw datasets here (not committed to Git due to size).

## Expected layout

```text
data/
├── raw/
│   ├── Divar.csv
│   └── iran_city_classification.csv   # optional, for statistical analysis
└── processed/                          # generated by pipelines
    ├── price_train.parquet
    └── price_val.parquet
```

## Setup

1. Download `Divar.csv` from your data source.
2. Copy it to `data/raw/Divar.csv`, or set `DIVAR_CSV` in `.env` (see `.env.example`).
3. Run preparation:

```bash
pip install -e .
divar-prepare-price
```
